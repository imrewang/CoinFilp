# CoinFilp

Qt翻金币小游戏

## 一、简介

翻金币项目是一款经典的益智类游戏，我们需要将金币都翻成同色，才视为胜利。

### （一）页面切换问题

本项目中使用Qt创建不同窗口，通过不同窗口的hide和show来进行页面切换

### （二）#Qt使用问题

在此项目中，使用Qt以不同页面为主体对象，同时兼顾可以复用的按钮等

主要是以页面为中心
